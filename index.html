BSA / unetdeux⭕

L'objectif du jeu est d'apprendre à manipuler des indices de matrices.


Comment jouer ?


Placer avec des coordonnées en colonne et en ligne des point "1" ou "2"
Pour gagner c'est les même règle que le morpion.


Comment le compiler et l'executer

Sous Linux :

Installez git et gcc avec votre gestionnaire de paquet

Clonez le repo unetdeux :

git clone https://github.com/benstitousofiane/unetdeux


Compilez le avec gcc :

cd unetdeux


gcc unetdeux.c -o unetdeux.bin


Lancer le binaire :

./unetedeux.bin


Sous Windows :

Installez TCC de fabrice bellard

Glissez le fichier unetdeux.c dans le dossier tcc

Puis compilez le :

tcc.exe unetdeux.c -o unetdeux.exe


Et executer le :

unetdeux.exe


Sous macOS

Installez Xcode, ça installera automatiquement git et gcc

Puis suivez les même étapes que la compilation sous GNU/Linux.

quitter le jeu
